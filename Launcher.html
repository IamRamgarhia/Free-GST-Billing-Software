Server won't start? Open this for help

Try these in order:

  1. Click ▶️ Start Server above. If your browser asks to allow freegstbill://, click Allow.
  2. If nothing happens after 30 seconds, open the install folder and double-click Start FreeGSTBill.bat directly. A small black window should appear briefly.
  3. If you see "Node.js is not installed", run Install FreeGSTBill.bat first.
  4. If the port is busy: open data\port.txt in Notepad to see which port the app is on, or delete that file and restart to let the server pick a fresh port.
  5. Still stuck? Email Contact@dicecodes.com or open a GitHub issue.
